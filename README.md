# About the Project

The **Hybrid Amazon Product Recommendation System** is a content-based recommendation system that suggests similar Amazon products by combining **Natural Language Processing (NLP)** and **Machine Learning** techniques.

This project analyzes product information such as **product name, main category, sub-category, ratings, number of ratings, price, and discount percentage** to identify products that are most similar to a selected item.

To improve recommendation quality, **Sentence Transformers** are used to generate semantic text embeddings, which are combined with scaled numerical features to create a **hybrid feature representation**. The recommendations are then generated using the **K-Nearest Neighbors (KNN)** algorithm with **Cosine Similarity**.

The project follows a complete end-to-end machine learning workflow, including **Data Cleaning, Exploratory Data Analysis (EDA), Feature Engineering, Model Training, and Streamlit Deployment**.

---

## 🚀 Key Features

- 🔍 Search and recommend similar Amazon products
- 🤖 Hybrid recommendation using NLP and Machine Learning
- 🧠 Sentence Transformer embeddings for semantic understanding
- 📊 Feature engineering using numerical product attributes
- ⚡ Fast recommendations using K-Nearest Neighbors (KNN)
- 🖼️ Product images and direct Amazon product links
- 💻 Interactive and user-friendly Streamlit interface

## 🎯 Project Objective

The primary objective of this project is to build an end-to-end product recommendation system capable of suggesting relevant products based on their semantic and numerical similarities. It demonstrates the complete machine learning pipeline, from data preprocessing and feature engineering to model deployment, making it a practical portfolio project for Data Science and Machine Learning applications.
